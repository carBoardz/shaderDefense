# 场景设置说明 - 如何把两个实验分开

## 方法一：创建两个独立场景（推荐）

### 步骤 1：保存当前场景为实验 1

1. 在 Unity 中，`File` → `Save As`
2. 命名为 `Experiment1_Phong.unity`
3. 保存到 `Assets/Scenes/` 文件夹

### 步骤 2：设置实验 1（Phong 多光源）

在这个场景中只保留：
- 1 个 Sphere + 地面
- Directional Light + 3 个附加光源（红、绿、蓝）
- LightController 脚本
- ForwardRendering_Phong 材质

删除所有与实验 2 相关的东西。

### 步骤 3：创建实验 2 场景

1. `File` → `New Scene`
2. `File` → `Save As`
3. 命名为 `Experiment2_Lambert.unity`
4. 保存到 `Assets/Scenes/` 文件夹

### 步骤 4：设置实验 2（Lambert 对比）

在这个场景中设置：
- 2 个 Sphere（左右各一个）
- Directional Light
- LambertCompareController 脚本
- Lambert_VertexMat 和 Lambert_PixelMat 材质

### 步骤 5：切换场景

在 `File` → `Open Scene` 中选择要打开的场景。

---

## 方法二：在同一场景中使用分层显示

如果你想在同一项目中方便切换，可以使用 GameObject 分组：

### 步骤 1：创建两个空物体分组

1. 创建空物体，命名为 `Experiment1_Phong`
2. 创建另一个空物体，命名为 `Experiment2_Lambert`

### 步骤 2：把相关物体拖到分组内

- 把实验 1 的所有物体拖到 `Experiment1_Phong` 下
- 把实验 2 的所有物体拖到 `Experiment2_Lambert` 下

### 步骤 3：通过开关显示/隐藏

- 做实验 1 时，激活 `Experiment1_Phong`，停用 `Experiment2_Lambert`
- 做实验 2 时，相反

## 实验报告使用说明

两个实验报告文件位置：
1. `Assets/实验报告1_Phong多光源前向渲染.md`
2. `Assets/实验报告2_Lambert漫反射对比.md`

### 如何复制到 Word

1. 在 VS Code 或任意 Markdown 编辑器中打开 .md 文件
2. 全选内容（Ctrl+A）
3. 复制（Ctrl+C）
4. 打开 Word，粘贴（Ctrl+V）
5. 在 Word 中调整格式（字体、标题、表格等）
6. 替换 `[请填写]` 为你的实际数据
7. 插入实验截图
