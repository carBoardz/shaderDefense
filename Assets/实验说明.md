# Unity 前向渲染与阴影实验

## 实验目的

1. 掌握 Unity 前向渲染路径的 ForwardBase 与 ForwardAdd 双 Pass 工作机制，理解多光源光照叠加原理
2. 能够在 Shader 中正确实现光照衰减、阴影投射与接收，提升场景真实感与空间层次
3. 结合光源类型、渲染模式与宏指令，完成可落地、工程化的光照 Shader 开发

## 文件结构

```
Assets/
├── Shaders/
│   ├── ForwardRendering_BlinnPhong.shader  # 实验1：双 Pass 多光源 Blinn-Phong
│   ├── NormalMapping_Forward.shader        # 实验2：法线贴图 + 前向渲染 + 阴影
│   └── AlphaTest_Shadow.shader             # 实验3：AlphaTest 镂空物体 + 双面阴影
├── Scripts/
│   └── LightModeSwitcher.cs                # 运行时切换光源 Render Mode
└── Materials/                              # 创建材质使用
```

## 实验内容

### 实验1：双 Pass Blinn-Phong 光照 Shader

**Shader**: `Custom/ForwardRendering_BlinnPhong`

**功能特性**：
- ForwardBase Pass：处理主光源 + 环境光 + 阴影
- ForwardAdd Pass：处理附加光源（混合模式 One One）
- 支持多个光源叠加
- 内置阴影支持（SHADOW_ATTENUATION）

**使用方法**：
1. 创建材质并选择此 Shader
2. 赋值主纹理
3. 在场景中添加多个光源
4. 附加 LightModeSwitcher 脚本到任意 GameObject
5. 运行时按空格键切换光源 Render Mode，使用 Frame Debugger 观察 Pass 调用

### 实验2：法线贴图 + 阴影 Shader

**Shader**: `Custom/NormalMapping_Forward`

**功能特性**：
- 切线空间法线贴图
- 金属度参数控制（可模拟金属、石材等质感）
- 双 Pass 前向渲染支持
- 正确的阴影投射与接收
- 切线→世界空间矩阵构建

**使用方法**：
1. 创建材质并选择此 Shader
2. 赋值主纹理和法线贴图
3. 调整 Normal Scale、Metallic、Gloss 参数
4. 验证多光源下法线细节与阴影的正确交互

### 实验3：AlphaTest 镂空物体 Shader

**Shader**: `Custom/AlphaTest_Shadow`

**功能特性**：
- AlphaTest 透明度测试（clip 函数）
- 自定义 ShadowCaster Pass
- 双面渲染（Cull Off）

- 解决背光面不投射阴影问题

**使用方法**：
1. 创建带 Alpha 通道的纹理（树叶、栅栏等）
2. 创建材质并选择此 Shader
3. 调整 Cutoff 阈值
4. 验证阴影与物体透明形态一致

## LightModeSwitcher 使用说明

将脚本挂载到场景任意 GameObject，在 Inspector 中设置需要控制的 Lights 数组。

| 按键 | 功能 |
|------|------|
| 空格键 | 循环切换光源 Render Mode（Auto → ForcePixel → ForceVertex） |

## 关键技术点

### ForwardBase vs ForwardAdd

- **ForwardBase**：每个物体渲染一次，处理主光源 + SH 光源 + 光照探针
- **ForwardAdd**：每个附加光源渲染一次，使用混合叠加

### 重要宏指令

```hlsl
#pragma multi_compile_fwdbase              // ForwardBase 变体
#pragma multi_compile_fwdadd_fullshadows   // ForwardAdd + 阴影
```

### 阴影相关

- `SHADOW_COORDS(idx)`：声明阴影坐标
- `TRANSFER_SHADOW(o)`：传递阴影坐标
- `SHADOW_ATTENUATION(i)`：采样阴影衰减
- 自定义 ShadowCaster Pass 用于特殊需求（如 AlphaTest）

### 切线空间

- TBN 矩阵：Tangent → Binormal → Normal
- 在顶点着色器中预计算切线→世界矩阵
- 片元着色器中采样法线贴图后转换到世界空间
